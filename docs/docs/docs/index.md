# Checkpost

Checkpost is an open-source osquery manager. It enrolls hosts, serves osquery configuration, runs scheduled and live queries, evaluates policies, and collects results.

## Where to start

Use [setup](setup/) to start the server, configure HTTPS, and enroll a host. The [feature guides](features/) follow the pages in the web interface. The [GitOps guide](gitops/) documents resources managed with `checkpost apply`.

## Current limits

Checkpost is still in active development. There will be breaking changes.
