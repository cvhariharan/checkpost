Manage osquery with ease.

Checkpost is an open-source osquery manager in a single binary.

Features

Easy Enrollment

Bootstrap scripts for macOS, Linux, and Windows hosts.

Inventory

Track hosts, owners, asset IDs, groups, and last-seen state.

Scheduled Queries

Run queries on a schedule and send results to multiple backends for storage.

Policies

Use policies to perform checks across enrolled hosts for device posture.

YARA

Scan files for YARA matches.

Ad-hoc Queries

Run on-demand SQL queries against hosts.

Alerts

Configure alerts for policy failures

Access Control

Role based access control with SSO support.

GitOps

Use checkpost CLI to manage resources defined in YAML.
